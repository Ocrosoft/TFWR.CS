
# TFWR.CS

TFWR.CS 是一个用于《编程农场》游戏的 C# 到类 Python 代码的转换工具。它让你可以用 C# 编写游戏脚本，并自动生成可在游戏中运行的代码。

## 主要功能

- 使用 C# 编写《编程农场》游戏脚本
- 自动将 C# 代码转换为游戏支持的类 Python 代码

## TFWR.CS.REF

在编写游戏脚本时，需要引用 TFWR.CS.REF 包。该包包含了游戏内置方法的声明，便于代码补全和类型提示。

## 使用须知

- 本项目处于开发阶段，仅供娱乐。
- 由于《编程农场》脚本语法较为简单，许多 C# 语法和特性无法被正确转换。
- 不支持直接调试。若需单步运行或调试，请在游戏内对生成的代码进行调试。
